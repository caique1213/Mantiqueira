# 15 — Comportamento Mobile e Desktop

## Princípio

Responsividade real.

## Desktop

Aproveitar largura para:

- mapa amplo;
- painéis laterais;
- filtros persistentes;
- timeline lado a lado;
- comparação.

## Mobile

Prioridades:

- toque;
- legibilidade;
- navegação simples;
- formulários em etapas;
- câmera;
- bottom sheets;
- mapa com zoom/scroll.

## Mapa no mobile

Não reduzir até ficar ilegível.

Opções aceitáveis:

- zoom;
- pan;
- scroll horizontal/vertical;
- modo foco.

## Bateria no mobile

- zoom;
- pan;
- pinch;
- tap target grande;
- painel inferior.

## Modos

O sistema antigo possuía seletor PC/Mobile.

A nova versão deve ser responsiva automaticamente.

Um override manual pode existir apenas como ferramenta de diagnóstico/preferência, não como base da responsividade.
