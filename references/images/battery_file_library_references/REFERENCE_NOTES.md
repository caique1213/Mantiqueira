# Referências originais da bateria — índice

## Conteúdo essencial extraído das referências

### Desenho-base

- Vista lateral.
- Círculos = motores.
- Cinza = carrinho de ração.
- Azul = esteira de nylon.
- Preto = bateria/gaiolas.
- Laranja = esteira branca.
- Vermelho = elevador de ovos.
- Esteira preta transversal no fundo.

### Correções posteriores do usuário

- São apenas 2 motores de esteira de nylon por bateria.
- Cada motor alimenta 3 esteiras de nylon.
- São 2 motores de esteira branca por bateria.
- Cada motor alimenta 3 esteiras brancas.
- A esteira preta não precisa aparecer fisicamente na vista lateral.
- Apenas o motor da esteira preta precisa aparecer.
- O carrinho de ração deve ser representado como retângulo/estrutura da parte superior até a parte inferior útil da bateria.
- O carrinho percorre a bateria distribuindo ração.
